# verilog-calculator
 # 🔢 Verilog Calculator (4-bit ALU)

A 4-bit Arithmetic Logic Unit (ALU) implemented using Verilog HDL.
Performs basic arithmetic and logical operations.

## ⚡ Operations Supported
| Operation | Symbol | Description |
|-----------|--------|-------------|
| Addition | ADD | Adds two 4-bit numbers |
| Subtraction | SUB | Subtracts two 4-bit numbers |
| AND | AND | Bitwise AND operation |
| OR | OR | Bitwise OR operation |

## 📁 Project Structure
verilog-calculator/
├── src/
│   ├── alu.v          # Main ALU module
│   ├── adder.v        # 4-bit adder
│   └── subtractor.v   # 4-bit subtractor
└── testbench/
└── alu_tb.v       # Testbench for ALU

## 🛠️ Tools Used
- Verilog HDL
- ModelSim / Xilinx Vivado
